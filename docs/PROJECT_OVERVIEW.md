# DeepSceneLoc - Project Overview

## Project Title
Visual Scene Understanding for Image-Based Location Estimation Using Deep Learning

## Project Goal
Develop a deep learning system that estimates location categories (Urban, Rural, Coastal, Mountain, Forest) from images without using GPS, EXIF data, or metadata - using only visual scene understanding.

## Constraints
- Only free and open-source tools
- No commercial libraries
- Free datasets (Places365)
- Free GPU (Google Colab)
- No paid services

## Key Features
- Semantic location categorization
- Visual scene analysis
- Deep learning-based classification
- Comparative model analysis
- Scene embedding representation

## Project Duration
Two semesters (30% current semester, 70% next semester)

## Target Location Categories
1. Urban - cities, downtown areas, streets
2. Rural - farmland, countryside, villages
3. Coastal - beaches, seaside, harbors
4. Mountain - peaks, slopes, highland areas
5. Forest - wooded areas, dense vegetation

## Success Criteria
- Baseline model accuracy above 70%
- Advanced model accuracy above 85%
- Comprehensive error analysis
- Research-quality documentation
- Viva-ready presentation

## Key Technologies
- PyTorch/TensorFlow (Deep Learning)
- ResNet-50 (Baseline)
- EfficientNet (Advanced)
- Vision Transformer (Advanced)
- Google Colab (Training)
- Places365 Dataset
